# 说明

GenController是一个功能强大的代码生成和管理工具，专注于处理代码生成、表结构管理以及数据表操作。它支持多种操作，包括查询、导入、修改、删除、预览和下载代码。通过这些功能，用户可以高效地管理和操作数据表，同时生成所需的代码，提升开发效率。
